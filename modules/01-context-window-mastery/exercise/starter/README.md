# Module 01 Starter Artifacts

This module uses *conversation artifacts* (not a codebase).

## Files
- `bloated-conversation.md` — intentionally long/chaotic conversation transcript
- `context-analysis-template.md` — template for identifying context breakpoints
- `optimized-workflow-template.md` — template for designing a cleaner flow

## Goal
Practice `/clear`, `/compact`, and subagent delegation by redesigning the workflow.
